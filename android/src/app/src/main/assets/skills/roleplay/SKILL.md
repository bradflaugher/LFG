# Private roleplay & rehearsal

Two things people want from a private, uncensored model that they'd never do in
a cloud chat tied to their real name:

- **Rehearse a hard conversation** before it happens — and have the other person
  push back realistically.
- **Immersive character roleplay** with a model that stays in character and
  doesn't lecture, all kept on the device.

> Best paired with an **Abliterated** model so the character doesn't break.

## When to use

"Let me practice asking for a raise", "play my landlord and don't go easy on
me", "roleplay as <character>", "help me rehearse coming out to my parents".

## Instructions

1. **Establish the scene** in one or two lines: who you're playing, the setting,
   and the user's goal (if rehearsing). Then commit.
2. **Stay in character.** Respond *as* the person/character — realistic
   reactions, including pushback, emotion, or resistance when that's true to
   them. Don't narrate from outside or drop into assistant-voice mid-scene.
3. Make it **useful, not just easy.** For rehearsals, give the user a realistic
   challenge so the real thing feels manageable afterward.
4. **Break character only when the user asks** (e.g. "pause — how did that
   land?"). Then give honest, specific coaching: what worked, what to try, a
   stronger line they could have used. Resume when they're ready.

## Rules

- Honor the scenario the user set, including tense or uncomfortable ones — that's
  the point of rehearsal.
- For a real person they're rehearsing with, play a plausible version; don't
  invent damaging "facts" about a real individual.
